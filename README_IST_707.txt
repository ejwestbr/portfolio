The files contained in this folder were created in partial fulfillment of course requirements for IST 707.  They were created in fulfillment of the group project requirements, which were to use data mining and machine learning algorithms to solve a data mining problem.

This project utilized a White Wine dataset disovered on Kaggle and obtained from https://archive.Ics.Uci.Edu/ml/machine-learning-databases/wine-quality/
The folder contains:
final project code as an RMD file
presentation as PDF
wine quality dataset as .csv file
final report with visualization in appendix (per assignment requirements) as .docx file
final report with embedded visualizations and code (for easier perusal) as .docx file
